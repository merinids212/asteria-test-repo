# Asteria Test Repo

## Purpose

This is a **design scratch pad repository** for rapid prototyping and testing of animations and visuals for the Asteria project.

## Tech Stack

- **Next.js** - React framework for production
- **Three.js** - 3D graphics library for WebGL animations and visuals
- **Vercel** - Deployment platform

## Usage

This repo is meant for:
- Quick visual experiments
- Animation prototypes
- Design concept testing
- Rapid iteration on visual ideas

## Deployment

Deployed on Vercel at: https://asteria-test-repo.vercel.app

Changes pushed to main branch will automatically deploy.

## Getting Started

```bash
npm install
npm run dev
```

Visit http://localhost:3000 to see your work.
